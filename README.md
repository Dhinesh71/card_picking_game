cd# Setup Instructions

## Prerequisites
- Node.js (v18+)
- npm

## Usage

1. **Install Backend Dependencies**
   ```bash
   cd backend
   npm install
   ```

2. **Start Backend**
   ```bash
   npm run dev
   # Server runs on port 3000
   ```

3. **Install Frontend Dependencies** (In a new terminal)
   ```bash
   cd frontend
   npm install
   ```

4. **Start Frontend**
   ```bash
   npm run dev
   # App runs on http://localhost:5173
   ```

## Admin Panel
- Open the exact URL shown in the terminal.
- Click "Admin Panel" in the top right.
- Use "User ID" `guest` to control your current session.
